## 📋 What is slides-grab?

slides-grab is a simple tool for creating and editing slide presentations. It works with Claude Codex and Codex formats. It helps you make slides faster and ensures they look consistent. You can write your slide content, fix common errors, and organize everything in one place.

This tool is designed for anyone who needs to build presentations without complicated software. You don't need to know any programming to use it.

## 📂 How to Use slides-grab

### Creating New Slides

- Click the “New Slide” button to add a slide.
- Choose slide types like title slide, bullet points, or image slide.
- Enter your text or add images in the editor area.
- Slides are saved automatically in your project.

### Editing Existing Slides

- Open your slide project file by clicking “Open Project” from the menu.
- Select the slide you want to modify.
- Edit content using the editor panel.
- Save your changes with the “Save” button.

### Using the Linter

- The linter checks for spelling errors and common formatting problems.
- After editing, click the “Check Slides” button.
- The tool lists warnings and errors with suggestions.
- Fix the issues and run the linter again to confirm.

### Exporting Your Presentation

- Click “Export.”
- Choose export format—usually PPT or PDF format.
- Slides will be saved in the location you select.

## 🛠 Troubleshooting

- **The installer won’t start:**  
  Make sure your Windows is up to date. Right-click the installer and choose “Run as administrator.”

- **slides-grab crashes or freezes:**  
  Close the app and restart your computer. Open slides-grab again.

- **Cannot open or save projects:**  
  Check if your files are stored in accessible folders. Avoid using network drives or removable USB drives during editing.

- **Linter reports errors I don’t understand:**  
  Review messages carefully. The tool highlights exactly where the issue is.

- **Exported slides don’t open in my presentation software:**  
  Use the latest PowerPoint or compatible app. If problems continue, export again and try PDF format.

- For more support, open an issue on the GitHub repository.
